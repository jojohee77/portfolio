# 🎯 이미지 위치 조정 가이드

낱개 텍스트 이미지의 위치를 정확하게 맞추는 방법입니다.

## 📋 방법 1: 브라우저 개발자 도구 (가장 쉬움!)

### 1단계: 개발 서버 실행
```bash
npm run dev
```
브라우저에서 http://localhost:5174 열기

### 2단계: 개발자 도구 열기
- **Windows**: `F12` 또는 `Ctrl + Shift + I`
- **Mac**: `Cmd + Option + I`

### 3단계: 이미지 선택
1. 개발자 도구에서 `Elements` 탭 클릭
2. 왼쪽 상단의 선택 도구(화살표) 아이콘 클릭
3. 조정하고 싶은 글자 이미지 클릭

### 4단계: 실시간 위치 조정
오른쪽 `Styles` 패널에서 값을 직접 수정:

```css
/* 예시 */
left: 8%;      /* ← 이 값을 바꿔보세요! */
top: 5%;       /* ← 이 값을 바꿔보세요! */
width: 240px;  /* 크기 조정 */
rotate: -5deg; /* 회전 각도 */
```

### 5단계: 코드에 반영
원하는 위치를 찾았다면, 그 값을 복사해서 코드에 붙여넣기:

**파일 위치**: `src/components/sections/HeroSection.tsx`

```typescript
const portfolioLetters = [
  [
    { 
      src: '/images/intro-txt-1-1.png', 
      alt: 'P', 
      left: '8%',    // ← 여기 수정
      top: '5%',     // ← 여기 수정
      width: 280,    // ← 여기 수정
      rotate: -5     // ← 여기 수정
    },
    // ... 나머지 글자들
  ]
]
```

## 📐 방법 2: Figma에서 좌표 추출

### 1단계: Figma에서 좌표 확인
1. Figma에서 글자 이미지 선택
2. 오른쪽 패널에서 `X`, `Y` 좌표 확인
3. `W` (width), `H` (height) 확인

### 2단계: 퍼센트로 변환
```
left % = (X 좌표 / 전체 캔버스 width) × 100
top % = (Y 좌표 / 전체 캔버스 height) × 100
```

**예시:**
- 캔버스 크기: 1920 × 1080px
- P 글자 위치: X=150px, Y=80px
- 계산: left = (150/1920)×100 = 7.8% ≈ 8%
- 계산: top = (80/1080)×100 = 7.4% ≈ 7%

## 🖼️ 방법 3: 이미지 크기가 너무 작은 경우

### 해결 방법 A: 더 큰 이미지 내보내기
Figma/Photoshop에서:
- 내보내기 배율을 `2x` 또는 `3x`로 설정
- 예: 원본 200px → 2x = 400px

### 해결 방법 B: 코드에서 크기 키우기 (현재 적용됨)
```typescript
// width 값을 크게 설정
{ src: '/images/intro-txt-1-1.png', alt: 'P', width: 280 } // ← 280px
```

## 🎨 반응형 크기 조정

현재 각 화면 크기별로 자동 조정됩니다:

| 화면 크기 | 글자 크기 |
|----------|----------|
| 모바일 (~ 640px) | 100px |
| 태블릿 (640px ~) | 130px |
| 데스크톱 (768px ~) | 160px |
| 대형 (1024px ~) | 200px |
| 초대형 (1280px ~) | 240px |

수정하려면 `className` 부분을 변경:
```typescript
className="absolute cursor-pointer 
  w-[100px]    // 모바일
  sm:w-[130px] // 태블릿
  md:w-[160px] // 데스크톱
  lg:w-[200px] // 대형
  xl:w-[240px] // 초대형
"
```

## 💡 유용한 팁

### 1. 그리드 오버레이 사용
개발자 도구에서 CSS Grid를 시각화:
- `...` 메뉴 → `More tools` → `CSS Grid debugger`

### 2. 한번에 모든 글자 크기 조정
`width` 값을 일괄 변경:
```typescript
// 모든 width에 1.2배 곱하기
width: 280 * 1.2  // = 336
```

### 3. 위치 미세 조정 단축키
개발자 도구에서 값 선택 후:
- `↑` / `↓`: 1씩 증가/감소
- `Shift + ↑/↓`: 10씩 증가/감소

### 4. 여러 화면에서 테스트
- 개발자 도구 → `Toggle device toolbar` (Ctrl+Shift+M)
- iPhone, iPad, Desktop 등 다양한 화면 크기 테스트

## 🔧 현재 설정된 값

```typescript
// P - 좌측 상단, 은색, 파란 별 옆
{ left: '8%', top: '5%', width: 280, rotate: -5 }

// O - 중앙 상단, 주황색
{ left: '38%', top: '3%', width: 240, rotate: 0 }

// R - 우측 상단, 초록 격자, 회색 웃는 얼굴
{ left: '65%', top: '8%', width: 260, rotate: 8 }

// T - 좌측 중앙, 노란색
{ left: '5%', top: '38%', width: 240, rotate: -3 }

// F - 중앙, 은색
{ left: '35%', top: '36%', width: 220, rotate: 5 }

// O - 우측 중앙, 주황색, 바늘
{ left: '62%', top: '40%', width: 250, rotate: -8 }

// L - 좌측 하단, 초록 격자
{ left: '10%', top: '68%', width: 260, rotate: 2 }

// I - 중앙 하단, 파란 웃는 얼굴
{ left: '40%', top: '70%', width: 120, rotate: 0 }

// O - 우측 하단, 노란색
{ left: '68%', top: '68%', width: 250, rotate: 0 }
```

## 📝 추천 작업 순서

2. ✅ **브라우저 개발자 도구 열기** (F12)
3. ✅ **한 글자씩 선택해서 위치 조정**
4. ✅ **만족스러운 값을 메모장에 복사**
5. ✅ **모든 글자 조정 완료 후 코드에 한번에 반영**
6. ✅ **다양한 화면 크기에서 테스트**
